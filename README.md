# Network Infrastructure Engineering

Design and implementation of enterprise network infrastructure, progressing from foundational Layer 2 networking to advanced routing and hybrid integration.

## Overview

This repository follows a versioned approach to building and evolving a network system. Each version represents a complete stage of development, focusing on practical design, implementation, and validation.


## Structure

* **v1 – Small Business Network**
  VLANs, inter-VLAN routing, and core services

* **v2 – Enterprise Enhancements** *(planned)*
  Redundancy, STP, EtherChannel

* **v3 – Routing Integration** *(planned)*
  OSPF and scalable network design

* **v4 – Hybrid Network** *(planned)*
  Cloud and hybrid connectivity

## Goal

To demonstrate practical network engineering skills through structured, real-world system development.
