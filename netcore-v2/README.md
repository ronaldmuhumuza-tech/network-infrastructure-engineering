# Netcore v2

Netcore v2 is a systems engineering infrastructure laboratory developed to explore the design, deployment and operation of modern enterprise infrastructure platforms.

The project combines networking, Linux administration, wireless infrastructure, monitoring, automation and cloud connectivity within a controlled EVE-NG environment.

The primary objective is to develop practical systems engineering capability through hands-on implementation of interconnected infrastructure components rather than studying individual technologies in isolation.

## Areas of Focus

* Network Infrastructure
* Linux Systems Administration
* Enterprise Wireless
* Infrastructure Monitoring
* Network Automation
* Cloud Connectivity
* Systems Engineering Principles
